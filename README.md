# CGMS Project

This is a basic full stack web application built using Django and React.

## Features
- User interface built with React
- Backend developed using Django
- Handles basic data operations
- Simple integration between frontend and backend

## Tech Used
- Frontend: React (Vite)
- Backend: Django
- Database: SQLite3 (default Django database)

## Project Structure
- frontend/ contains the React application
- backend/ contains the Django server

## How to Run
1. Clone the repository
2. Go to the backend folder and install requirements
3. Run Django server
4. Go to frontend folder and install dependencies
5. Start the React development server
